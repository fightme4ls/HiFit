Database Implementation

For the establishment of the database infrastructure, a deliberate choice was made to leverage MySQL, a robust and widely-adopted relational database management system. The initial phase involved meticulous planning and conceptualization of the table structure. Once the blueprint for the database was defined, tables were systematically designed, complete with their respective attributes. To ensure comprehensive accessibility for the entire team, a strategic decision was made to deploy the database on the Google Cloud platform. The process involved establishing a seamless connection between MySQL Workbench and Google Cloud, subsequently initializing the database within this cloud environment. This methodical approach not only facilitated the creation of a well-organized and efficient database but also streamlined collaborative efforts by enabling remote access for the entire project team.
